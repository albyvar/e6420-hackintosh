DELL E6420 - HACKINTOSH

Quick tutorial + All tools needed + Triple boot

Dell E6420 Specifications

  • CPU: Intel Core i5 - 2520M
  • RAM: 4GB
  • GPU: Intel HD 30000
  • WIFI: Intel Centrino Advanced-N 6205 - NOT SUPPORTED BY OS X - I replaced it with a Broadcom BCM43225

What's working in OSX:

  • Everything, but:
  • Touchpad is recognized as mouse (so cursor speed will be a bit slow)
  • HDMI Audio is buggy

Required things

  • A pen drive with macOS Sierra installer + Clover bootloader
  • You may want to use my EFI partition, already configured for this model and for triple boot: download it (here)
  • Your HDD should be clean if you want to triple boot (clean it by booting the windows installer in uefi mode and through cmd run "clean"
  • A pen drive with Windows 7/8/10 UEFI (that's important) installer
  • A pen drive with Ubuntu 16.04 UEFI installer (other linux distros should also be good)

Procedure (Windows)

  • First of all, we need to install windows on our freshly cleaned hdd drive.
  • Boot the installer, select custom install and install windows on the only partition it will show. (If windows tells you something like "to ensure windows will work properly, some other partitions will be created", ignore it and continue.
  • After windows is installed, boot it.
  • Now we need to resize/change our EFI partition, because Windows by default creates an EFI partition of 200mb, but OS X require a slightly larger EFI partition to be able to format partitions. If you skip this step, OS X won't be able to format partitions and will complain something like "Mediakit reports not enough space".
  • Follow this tutorial to resize/change EFI partition: (here)
  • Now, open start menu and search for "Create and edit disk partitions" and open the first result.
  • Select your windows partition and right click on it. Choose shrink volume.
  • Shrink it by your desired size.
  • Now repeat the process and create another partition (for ubuntu). Skip this step if you don't want to install ubuntu.

Procedure (macOS)

  • Boot the macOS installer usb.
  • Once booted, open Disk Utility and format the partition you want to use for macOS to "journaled HFS+ (macOS Extended)
  • Close disk utility and install macOS to the newly formatted partition.
  • Once installation is completed, reboot and, using the usb installer, boot to macOS.
  • Install Clover bootloader (download [here)
  • Mount EFI partition of USB and of HDD, and copy the EFI/Clover folder from the USB to the HDD EFI partition.
  • If you don't want to install Ubuntu or anything else, you're done.

Procedure (Ubuntu)

  • Boot the USB installer of Ubuntu
  • Choose the partition where you want to install Ubuntu in, and use it as filesystem (ext4 journaled)
  • IMPORTANT: Choose the current EFI partition (often called Windows Boot Manager by Ubuntu) as bootloader destination.
  • Install Ubuntu
  • Reboot, and you will see Clover has been replaced by GRUB. Don't worry, boot into BIOS, and set the priority back to Clover.
  • If your BIOS does not have an option to manage UEFI boot entries, use EasyUEFI program for windows
  • Once you set priority back to clover, reboot and enjoy your triple boot!
