DELL E6420 - HACKINTOSH

Quick tutorial + All tools needed + Triple boot

Dell E6420 Specifications

  • CPU: Intel Core i5 - 2520M
  • RAM: 4GB
  • GPU: Intel HD 30000
  • WIFI: Intel Centrino Advanced-N 6205 - NOT SUPPORTED BY OS X - I replaced it with a Broadcom BCM43225

What's working in OSX:

  • Everything, but:
  • Touchpad is recognized as mouse (so cursor speed will be a bit slow)
  • HDMI Audio is buggy

Required things

  • A pen drive with macOS Sierra installer + Clover bootloader
  • You may want to use my EFI partition, already configured for this model and for triple boot: download it (here)
  • Your HDD should be clean if you want to triple boot (clean it by booting the windows installer in uefi mode and through cmd run "clean"
  • A pen drive with Windows 7/8/10 UEFI (that's important) installer
  • A pen drive with Ubuntu 16.04 UEFI installer (other linux distros should also be good)
